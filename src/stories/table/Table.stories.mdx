## Custom table
 By default the gantt diagram doesn't display any table for showing the details of the <code>groups</code>
 
You can add one and customize the table of your <code><a href={encodeApiDocURIForSubpath("/class/src/timeline.js~Timeline.html")}>Timeline</a></code> by setting the <code>table</code> property. It should be a table from the <code>fixed-data-table-2</code> library. 
  
**IMPORTANT:** The cell renderer of your custom table should take in consideration that there could be empty rows (without any data behind) added by gantt just for filling in the remaining empty space
